DirectX 과제 2회차.

** 모두 프로젝트 따로 압축해서 첨부하고, 아래 내용 참고해서 스샷도 추가 해주세요 **

10.	HLSL 코드로 상황에 맞는 시맨틱(Semantic)을 적용해 정점 입력과 출력에 사용될 구조체를 정의할 수 있다(예: 입력)정점 위치, 출력)정점 위치, 정점 색상,)
	-> DirectX 쪽 Vertex(정점) 구조체 스샷 1장 / 쉐이더 쪽 정점 입력 구조체 스샷 1장 / 코드 실행 스샷 1장(내용은 상관 없음).

11.	FBX 파일로 저장된 메쉬를 로드해 화면에 그릴 수 있다.
	-> FBX 로더 함수내용 스샷 1장 / 캐릭터 로드한 결과 스샷 1장

12.	1인칭 시점에 적합한 카메라 클래스를 구현할 수 있다.
	-> 카메라 이동 전 스샷 1장 / 카메라 약간 이동 후 스샷 1장(앞에 스샷과 다른 시점)

13.	HLSL을 활용해 디퓨즈(난반사) 셰이더를 작성할 수 있다.
	-> 난반사(디퓨즈) 정점/픽셀 쉐이더 내용 스샷 1장씩(2장) / 구체 스샷 1장.

14.	HLSL을 활용해 스페큘러(정반사) 셰이더를 작성할 수 있다.
	-> 정반사(스페큘러) 정점/픽셀 쉐이더 내용 스샷 1장씩(2장) / 구체 스샷 1장.

15.	정점의 노멀 정보와 노멀맵 텍스처를 이용해 노멀매핑을 적용할 수 있다.
	-> 노멀맵 쉐이더 코드 스샷 1장씩(정점, 픽셀 쉐이더 각각) / 실행 결과 스샷

16.	툰 셰이딩 효과를 구현할 수 있다.
	-> 툰 쉐이더 코드 스샷 (정점, 픽셀 쉐이더 각각) / 실행 결과 스샷.
